star-wars-movies

Star Wars movies API app

To run project: npm start
To run tests: npm tests

Technologies used:

- nodejs 18.15.0
- axios: 1.4.0
- reduxjs/toolkit: 1.9.5
- react-redux: 8.1.2
- material: 5.14.4
- icons-material: 5.14.3
